## If Script Fails

**Common issues:**

1. **"Failed to clone"**
   - Check your GitHub username is correct
   - Verify you forked the repo
   - Check SSH keys: `ssh -T git@github.com`

2. **"CMake failed"**
   - Needs CUDA 13.0+
   - Run on L4 instance or local GPU machine

3. **"Build failed"**
   - Check CUDA_HOME is set
   - Verify nvcc is in PATH
